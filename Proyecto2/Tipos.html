Tipos de Servidores Web


Un servidor web opera en un ordenador aguardando las solicitudes de parte del navegador web de cualquier usuario, ofreciendo los datos solicitados para estructurar una página web o, en dado caso de algún inconveniente imprimir un mensaje de error.

REGRESAR

Servidores Estáticos:

Una página web estática es aquella la cual no permite la interacción con el usuario para alterar o modificar la información y/o estructura de la página. En este tipo de páginas los datos que muestra siempre ser exactamente la misma y su actualización depende de una alteración en código fuente o en base de datos que alimenta al sitio y no las reacciones que realice el usuario.


La característica de un servidor Estático es que esta depende de estar construida con una programación definida esto hace que un servidor estático se un servidor estático ya que al tener una programación definida o estática ya que el servidor solo mostrara una información única para cualquier usuario que solicite la información, sin ningún detalle diferente en ninguna ocasión.

Servidores Dinámicos:

Una página web dinámica es un documento en línea permite la interacción con el usuario la cual muestra diferencias si realiza una comparación de diferentes usuarios, gran parte de la información mostrada en la página dependerá de los usuarios con cada carga ofrece actualizaciones en tiempo real y se adapta a las necesidades de cada uno de los usuarios.


Una Característica de los Servidores Dinámicos es que su funcionamiento depende de comandos, debido a que los contenidos que se muestran en los servidores dinámicos esta sujeta a las interacciones que tiene con los usuarios y es bastante necesario que los programadores tengan que darle cierto grado de programación el cual deben optimizar el sitio en lenguajes como JavaScript o PHP para que la página genere los cambios necesarios y esperados por el sitio y el usuario
