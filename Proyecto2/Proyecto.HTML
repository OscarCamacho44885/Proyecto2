Definición

Un servidor web o también conocido como Servidor HTTP programa de software de comunicaciones el cual realiza una conexión entre el servidor en el que se encuentran establecidos datos a los cuales pueden ser solicitados y cualquier dispositivo que tenga un usuario, teniendo conexiones bidireccionales o unidireccionales, síncronas o asíncronas, con cualquier software que el usuario tenga. Se define como programas que Demian entre un usuario de internet y el servidor donde existe información que el usuario necesita solicitar y el servidor se encarga de mantener esa información disponible en cada momento en el que usuario la solicite.


¿Cuál es el proposito de los Servidores Web?

La función de un servidor web es la de mediar entre las solicitudes de los usuarios y los archivos que contienen la información solicitada en el computador en el que están almacenados, lo cual se necesita uno para publicar un sitio web y también para tener acceso a los datos que componen una página web de cualquiera.

  • El usuario ingresa una dirección web (URL) en su navegador y envía una solicitud al servidor web.
  • El servidor web (software) busca los archivos relevantes ya sea en el propio servidor (hardware) o en un servicio de hosting donde siempre están disponibles y en línea.
  • A continuación, los archivos se procesan según se desee y se envían según el protocolo de transferencia, es decir, según un conjunto de reglas que regulan la comunicación entre ordenadores.
  • El navegador recibe los archivos y compila el contenido de la página web mostrada al usuario.
  • Un servidor web se comunica con un navegador web mediante el protocolo de transferencia de hipertexto, también conocido como HTTP.
  • Además de HTTP, los servidores web también admiten SMTP (Protocolo simple de transferencia de correo) y FTP (Protocolo de transferencia de archivos), que se utilizan para correo electrónico, transferencia de archivos y almacenamiento.

    En definitiva, su tarea principal es almacenar y transmitir el contenido solicitado de un sitio web al navegador del usuario.
